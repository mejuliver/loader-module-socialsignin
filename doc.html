Load script

		$loader->load('socialsignin')->init(<config>,<url>);

Config

		[
			'console' => < 1 means all errors will be shown on console >,
			'fb' => [
				'client_id' => < fb app id >,
				'version' =>  < sdk version >,
				'url' => < url for backend submission >,
				'result' => < result function >,
			],
			'google' => [
				'client_id' => < google app id >,
				'url' => < url for backend submission >,
				'result' => < result function >,
			]
		]

Bind Social Classes and Identifier

Example

		<button class="socialSignin" data-type="fb">test</button>
		<button class="socialSignin" data-type="google">test</button>
